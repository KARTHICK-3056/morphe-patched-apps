# Morphe Patches PreBuild

Pre-patched YouTube and YouTube Music builds with Morphe Patches, available for both rooted and non-rooted devices.

## Installation Guide

### Rooted Devices

To prevent automatic updates from the Play Store, use one of these methods:

**Option 1: Zygisk Detach** (Magisk Module)
- [Download Zygisk Detach](https://github.com/j-hc/zygisk-detach/releases)

**Option 2: Update Locker** (Recommended - Requires LSPosed)
- [Download Update Locker](https://github.com/Xposed-Modules-Repo/ru.mike.updatelocker)

- **LSPosed Variants:**
    - [ReLSPosed](https://nightly.link/ThePedroo/ReLSPosed/workflows/core/master)
    - [LSPosed JingMatrix](https://nightly.link/JingMatrix/LSPosed/workflows/core/master)
    - [LSPosed-Irena](https://nightly.link/re-zero001/LSPosed-Irena/workflows/core/dev)

### Non-Rooted Devices

Rootless usage requires microG, so first download MicroG-RE.

- [Download MicroG-RE](https://github.com/WSTxda/MicroG-RE/releases)
